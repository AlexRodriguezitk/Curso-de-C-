Tutoriales de C++

Operadores

Los operadores en C++ permiten realizar operaciones matemáticas, lógicas y de manipulación de datos. Se dividen en varias categorías:

  • Operadores aritméticos: Se usan para realizar operaciones matemáticas básicas.
    • +: Suma
    • -: Resta
    • *: Multiplicación
    • /: División
    • %: Módulo (resto de la división)
    int a = 5;
int b = 2;
int suma = a + b; // Resultado: 7
int resta = a - b; // Resultado: 3
int modulo = a % b; // Resultado: 1
  • Operadores de comparación: Comparan dos valores y devuelven un valor booleano (verdadero o falso).
    • ==: Igual a
    • !=: Diferente de
    • >: Mayor que
    • <: Menor que
    • >=: Mayor o igual que
    • <=: Menor o igual que
    int x = 10;
int y = 20;
bool esIgual = (x == y); // Resultado: false
bool esMayor = (x > y);  // Resultado: false
  • Operadores lógicos: Se usan para combinar expresiones booleanas.
    • &&: AND lógico (ambos deben ser verdaderos)
    • ||: OR lógico (uno o ambos son verdaderos)
    • !: NOT lógico (invierte el valor)
    bool a = true;
bool b = false;
bool resultado = (a && b); // Resultado: false

Tipos de Datos

En C++, los tipos de datos determinan el tipo y tamaño de los valores que pueden almacenarse en una variable. Algunos de los tipos de datos más comunes son:

  • Tipos enteros: Almacenan números enteros sin parte decimal.
    • int: Número entero de 4 bytes.
    • short: Entero más pequeño (2 bytes).
    • long: Entero más grande (4 u 8 bytes dependiendo del sistema).
    int numero = 100;
short valorPequeño = 10;
long valorGrande = 1000000;
  • Tipos flotantes: Almacenan números con parte decimal.
    • float: Número de coma flotante de precisión simple (4 bytes).
    • double: Número de coma flotante de doble precisión (8 bytes).
    float pi = 3.14;
double e = 2.718281828;
  • Tipo booleano: Almacena valores lógicos (verdadero o falso).
    • bool esVerdadero = true;
  • Tipo carácter: Almacena un solo carácter.
    • char letra = 'A';

Condicionales

Las estructuras condicionales permiten ejecutar diferentes bloques de código en función de si se cumplen ciertas condiciones. Los condicionales en C++ incluyen:

  • if: Ejecuta un bloque de código si la condición es verdadera.
  • else: Se ejecuta si la condición en el if no es verdadera.
  • else if: Evalúa una nueva condición si la primera es falsa.
int numero = 5;
if (numero > 10) {
    std::cout << "Mayor que 10" << std::endl;
} else if (numero == 10) {
    std::cout << "Igual a 10" << std::endl;
} else {
    std::cout << "Menor que 10" << std::endl;
}

Bucles

Los bucles permiten ejecutar un bloque de código varias veces de forma repetitiva. Los principales tipos de bucles en C++ son:

  • for: Se utiliza cuando se conoce de antemano cuántas veces se ejecutará el bucle.
  • while: Se utiliza cuando la condición se evalúa antes de cada iteración.
  • do-while: Se ejecuta al menos una vez, ya que la condición se evalúa después de la primera iteración.
// Ejemplo de bucle for
for (int i = 0; i < 5; i++) {
    std::cout << "Iteración " << i << std::endl;
}

// Ejemplo de bucle while
int j = 0;
while (j < 5) {
    std::cout << "Iteración " << j << std::endl;
    j++;
}

// Ejemplo de bucle do-while
int k = 0;
do {
    std::cout << "Iteración " << k << std::endl;
    k++;
} while (k < 5);

Funciones

Las funciones en C++ son bloques de código reutilizables que realizan una tarea específica. Se definen con un tipo de retorno, un nombre y opcionalmente parámetros.

int sumar(int a, int b) {
    return a + b;
}

int main() {
    int resultado = sumar(5, 3);
    std::cout << "La suma es: " << resultado << std::endl;
    return 0;
}

En este ejemplo, la función sumar recibe dos parámetros a y b, y devuelve la suma de ambos. Luego, se llama a la función desde main para mostrar el resultado.
