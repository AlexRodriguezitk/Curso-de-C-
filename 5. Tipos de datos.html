Ejercicios de Tipos de Datos en C++

Documentación sobre Tipos de Datos en C++

Tipos de Datos Básicos

En C++, los tipos de datos básicos incluyen:

  • int: Enteros. Se utilizan para almacenar números enteros, ya sean positivos o negativos, sin decimales. Ejemplo: int edad = 25;
  • float: Números de punto flotante de precisión simple. Se utilizan para almacenar números con decimales. Ejemplo: float precio = 19.99;
  • double: Similar a float pero con el doble de precisión. Ejemplo: double distancia = 123.456;
  • char: Caracteres. Se utilizan para almacenar un solo carácter. Ejemplo: char letra = 'A';
  • bool: Valores booleanos. Se utilizan para almacenar true o false. Ejemplo: bool esMayor = true;
  • string: Cadenas de caracteres. En C++, las cadenas de caracteres se manejan con la clase string de la biblioteca estándar. Ejemplo: std::string nombre = "Juan";

Tipos de Datos Compuestos

Además de los tipos de datos básicos, C++ también ofrece tipos compuestos como:

  • array: Conjunto de elementos del mismo tipo. Los arrays en C++ tienen un tamaño fijo definido en el momento de su declaración. Ejemplo: int numeros[5] = {1, 2, 3, 4, 5};
  • ArrayList: En C++, no existe un tipo ArrayList directamente. Sin embargo, la biblioteca estándar proporciona std::vector, que es un contenedor que puede cambiar de tamaño dinámicamente y es más similar a un ArrayList de otros lenguajes como Java. Ejemplo: std::vector<int> listaNumeros = {1, 2, 3, 4, 5};

Ejercicios con Tipos de Datos en C++

  1. Ejercicio 1: Declara una variable int para almacenar la edad de una persona y muestra su valor.
  2. Ejercicio 2: Declara una variable float para almacenar el precio de un artículo y muestra su valor.
  3. Ejercicio 3: Escribe un programa que tome un número int del usuario y determine si es par o impar.
  4. Ejercicio 4: Declara un string para almacenar el nombre de una persona y muestra su valor.
  5. Ejercicio 5: Escribe un programa que almacene 5 números en un array y calcule su suma.
  6. Ejercicio 6: Declara un array de caracteres (char) para almacenar una palabra y muestra cada letra en una nueva línea.
  7. Ejercicio 7: Crea un std::vector de enteros, añade algunos elementos y muestra el tamaño del vector.
  8. Ejercicio 8: Escribe un programa que tome un array de enteros y devuelva el número máximo.
  9. Ejercicio 9: Usa un std::vector para almacenar 5 números y calcula su media.
  10. Ejercicio 10: Escribe un programa que invierta los elementos de un array de caracteres y muestre la cadena resultante.
