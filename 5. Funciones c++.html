Ejercicios de Funciones en C++

Documentación sobre Funciones en C++

¿Qué es una función?

Una función es un bloque de código que realiza una tarea específica. Las funciones permiten dividir un programa en partes más pequeñas y manejables. En C++, las funciones se definen usando la siguiente sintaxis básica:

tipo_de_retorno nombre_funcion(parámetros) {
    // Código de la función
    return valor; // si se requiere un valor de retorno
}

Tipos de Funciones

En C++, hay varios tipos de funciones:

  • Funciones sin valor de retorno (void): No devuelven ningún valor. Se utilizan para realizar acciones específicas.
  • Funciones con valor de retorno: Devuelven un valor al final de su ejecución. El tipo de valor devuelto debe coincidir con el tipo de retorno especificado al definir la función.
  • Funciones con parámetros: Reciben datos de entrada, los procesan y pueden devolver un resultado.
  • Funciones con parámetros por referencia: Permiten modificar los valores de las variables pasadas como argumentos.
  • Funciones recursivas: Se llaman a sí mismas dentro de su propio cuerpo, lo cual es útil para problemas que se pueden dividir en subproblemas similares.

Ejemplo de una Función con Retorno

Este ejemplo muestra una función que suma dos números y devuelve el resultado:

int suma(int a, int b) {
    return a + b;
}

Ejercicios con Funciones en C++

  1. Ejercicio 1: Escribe una función que tome dos números enteros y devuelva su suma.
  2. Ejercicio 2: Crea una función que determine si un número es par o impar. La función debe devolver un valor booleano.
  3. Ejercicio 3: Escribe una función que calcule el factorial de un número entero dado.
  4. Ejercicio 4: Crea una función que reciba un número entero y devuelva el número al cuadrado.
  5. Ejercicio 5: Escribe una función que determine si un número es primo.
  6. Ejercicio 6: Crea una función que reciba dos números y devuelva el mayor de ellos.
  7. Ejercicio 7: Escribe una función recursiva que calcule el n-ésimo término de la secuencia de Fibonacci.
  8. Ejercicio 8: Crea una función que convierta grados Celsius a Fahrenheit.
  9. Ejercicio 9: Escribe una función que reciba un array de enteros y su tamaño, y calcule la media de sus elementos.
  10. Ejercicio 10: Escribe una función que reciba una cadena de caracteres y devuelva la cadena invertida.
