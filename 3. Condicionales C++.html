Ejercicios de Condicionales en C++

Accede al Tutorial de C++

Si estás interesado en aprender los conceptos básicos de C++, como operadores, tipos de datos, condicionales, bucles y funciones, puedes acceder al tutorial completo en el siguiente enlace:

Ejercicio 1: Número Par o Impar

Escribe un programa que pida un número al usuario y determine si es par o impar.

Ejercicio 2: Positivo, Negativo o Cero

Escribe un programa que pida un número al usuario y determine si es positivo, negativo o cero.

Ejercicio 3: Mayor de Dos Números

Escribe un programa que pida dos números al usuario y determine cuál es mayor.

Ejercicio 4: Bisiesto o No

Escribe un programa que determine si un año introducido por el usuario es bisiesto o no.

Ejercicio 5: Notas de Examen

Escribe un programa que pida la calificación de un examen y muestre si es "Aprobado" o "Reprobado" (considerando que la nota mínima para aprobar es 60).

Ejercicio 6: Multiplicidad de Tres

Escribe un programa que pida un número y determine si es múltiplo de tres.

Ejercicio 7: Operación Aritmética

Escribe un programa que pida dos números y un operador (+, -, *, /) y realice la operación correspondiente.

Ejercicio 8: Categoría de Edad

Escribe un programa que pida la edad de una persona y determine su categoría: niño (0-12), adolescente (13-17), adulto (18-64), mayor (65+).

Ejercicio 9: Ordenar Tres Números

Escribe un programa que pida tres números y los muestre en orden ascendente.

Ejercicio 10: Validación de Contraseña

Escribe un programa que simule la validación de una contraseña. Si la contraseña introducida es "segura123", muestra "Acceso concedido"; de lo contrario, muestra "Acceso denegado".

Ejercicio 11: Días de la Semana

Escribe un programa que pida un número del 1 al 7 y use un switch para mostrar el día de la semana correspondiente.

/*
    Ejemplo de uso de switch en C++:

    switch(variable) {
        case valor1:

            break;
        case valor2:
            // código
            break;
        default:
            // código
    }

*/

Ejercicio 12: Meses del Año

Escribe un programa que pida un número del 1 al 12 y use un switch para mostrar el nombre del mes correspondiente.

Ejercicio 13: Tipo de Triángulo

Escribe un programa que pida tres lados de un triángulo y use un switch para determinar el tipo de triángulo (equilátero, isósceles, escaleno).

Ejercicio 14: Notas de Examen

Escribe un programa que pida la calificación de un examen y use un switch para mostrar si es "A", "B", "C", "D" o "F".

Ejercicio 15: Operaciones Matemáticas

Escribe un programa que pida dos números y un operador (+, -, *, /, %) y use un switch para realizar la operación correspondiente.

Ejercicio 16: Frutas

Escribe un programa que pida un número del 1 al 5 y use un switch para mostrar una fruta correspondiente a ese número.

Ejercicio 17: Menú de Comida

Escribe un programa que pida un número del 1 al 3 y use un switch para mostrar un plato de comida correspondiente.

Ejercicio 18: Grados Académicos

Escribe un programa que pida un número del 1 al 4 y use un switch para mostrar el grado académico correspondiente (1: Primaria, 2: Secundaria, etc.).

Ejercicio 19: Tipo de Vehículo

Escribe un programa que pida un número del 1 al 4 y use un switch para mostrar el tipo de vehículo correspondiente (1: Carro, 2: Motocicleta, etc.).

Ejercicio 20: Zodiaco Chino

Escribe un programa que pida un año y use un switch para mostrar el signo del zodiaco chino correspondiente.
