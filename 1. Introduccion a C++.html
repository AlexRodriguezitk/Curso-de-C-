Sintaxis Básica

En C++, un programa típico se organiza en funciones, siendo la función main() el punto de entrada. Aquí te explicamos cómo se estructura un programa básico:

#include <iostream>

// La función principal del programa
int main() {
    // Imprime un mensaje en la consola
    std::cout << "Hola, mundo!" << std::endl;
    return 0; // Fin del programa
}

Explicación:

  • #include <iostream>: Incluye la biblioteca estándar de entrada y salida.
  • int main(): La función principal que se ejecuta al iniciar el programa.
  • std::cout: Utilizado para imprimir mensajes en la consola.
  • std::endl: Agrega una nueva línea al final del mensaje.
  • return 0;: Indica que el programa terminó correctamente.

Entradas y Salidas

C++ utiliza std::cin para leer entradas del usuario y std::cout para mostrar resultados.

#include <iostream>

int main() {
    int numero;
    std::cout << "Introduce un número: ";
    std::cin >> numero; // Lee un número del usuario
    std::cout << "El número introducido es: " << numero << std::endl;
    return 0;
}

Explicación:

  • std::cin: Se utiliza para leer datos de entrada desde la consola.
  • >>: Operador de extracción que extrae datos del flujo de entrada (console).
  • std::cout << "Introduce un número: ": Imprime un mensaje solicitando al usuario que introduzca un número.

Operadores

C++ soporta varios tipos de operadores para realizar operaciones aritméticas, comparaciones y más.

#include <iostream>

int main() {
    int a = 5;
    int b = 10;
    std::cout << "Suma: " << (a + b) << std::endl;
    std::cout << "Multiplicación: " << (a * b) << std::endl;
    return 0;
}

Explicación:

  • a + b: Operador de suma que suma dos números.
  • a * b: Operador de multiplicación que multiplica dos números.
  • std::cout << "Suma: " << (a + b): Imprime el resultado de la suma.

Condicionales

Las estructuras condicionales permiten ejecutar diferentes bloques de código según ciertas condiciones.

#include <iostream>

int main() {
    int numero;
    std::cout << "Introduce un número: ";
    std::cin >> numero;

    if (numero % 2 == 0) {
        std::cout << "El número es par." << std::endl;
    } else {
        std::cout << "El número es impar." << std::endl;
    }
    return 0;
}

Explicación:

  • if (numero % 2 == 0): Comprueba si el número es divisible por 2 (es decir, si es par).
  • else: Ejecuta el bloque de código si la condición if no se cumple.
  • std::cout << "El número es par.": Imprime un mensaje si la condición es verdadera.

Bucles

Los bucles permiten ejecutar un bloque de código varias veces. C++ ofrece varios tipos de bucles.

#include <iostream>

int main() {
    for (int i = 1; i <= 5; ++i) {
        std::cout << "Número: " << i << std::endl;
    }
    return 0;
}

Explicación:

  • for (int i = 1; i <= 5; ++i): Un bucle for que inicia con i = 1 y se ejecuta mientras i sea menor o igual a 5.
  • ++i: Incrementa el valor de i en cada iteración del bucle.
  • std::cout << "Número: " << i: Imprime el valor de i en cada iteración.
