Ejercicio 1: Calculadora de Calificaciones

En este ejercicio, crearás un programa en C++ que calcula la calificación promedio de un estudiante y determina su calificación final basada en un sistema de letras (A, B, C, D, F). Utilizarás operadores para los cálculos, condicionales para las decisiones, bucles para la entrada de datos, funciones para modular el código y diferentes tipos de datos para manejar la información.

Documentación Paso a Paso:

  1. Definir las Variables: Define las variables necesarias, como un array para almacenar las calificaciones y un int para contar la cantidad de calificaciones.
  2. Función para Calcular el Promedio: Crea una función float calcularPromedio(int calificaciones[], int numCalificaciones) que recibe un array de enteros y el número total de calificaciones, y devuelve el promedio como un float.
  3. Función para Determinar la Letra de la Calificación: Crea una función char determinarLetra(float promedio) que recibe el promedio y devuelve la letra de la calificación correspondiente usando condicionales (if-else).
  4. Bucle para Entrada de Datos: Utiliza un bucle for para permitir que el usuario ingrese todas las calificaciones.
  5. Llamar a las Funciones: Llama a las funciones desde main() para calcular el promedio y determinar la letra de la calificación.
  6. Mostrar el Resultado: Muestra el promedio y la calificación final al usuario.

Ejercicio 2: Inventario de Tienda

Este ejercicio te guía para crear un sistema simple de inventario para una tienda. Utilizarás un std::vector para almacenar los nombres de los productos y sus precios, bucles para iterar sobre los productos, y funciones para agregar, eliminar y mostrar los productos en el inventario.

Documentación Paso a Paso:

  1. Definir las Estructuras de Datos: Define dos vectores (std::vector) para almacenar los nombres y precios de los productos.
  2. Función para Agregar un Producto: Crea una función void agregarProducto(std::vector<std::string> &nombres, std::vector<float> &precios) que toma los vectores por referencia y añade un nuevo producto.
  3. Función para Eliminar un Producto: Crea una función void eliminarProducto(std::vector<std::string> &nombres, std::vector<float> &precios, int index) que elimina un producto en una posición dada.
  4. Función para Mostrar Inventario: Crea una función void mostrarInventario(const std::vector<std::string> &nombres, const std::vector<float> &precios) que muestra todos los productos y sus precios.
  5. Bucle para el Menú Principal: En el main(), utiliza un bucle while para mostrar un menú que permita al usuario elegir entre agregar, eliminar o mostrar productos, o salir del programa.
  6. Implementar las Funciones: Asegúrate de que cada opción del menú llame a la función correspondiente y maneje los errores, como intentar eliminar un producto de un índice no válido.
