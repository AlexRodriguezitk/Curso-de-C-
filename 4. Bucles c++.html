Documentación sobre Bucles en C++

Bucle for

El bucle for se utiliza cuando se conoce de antemano el número de iteraciones. Su sintaxis básica es:

for (inicialización; condición; actualización) {
    // Código a ejecutar
}

Ejemplo: Imprimir números del 1 al 5

for (int i = 1; i <= 5; i++) {
    std::cout << i << std::endl;
}

Bucle while

El bucle while se utiliza cuando no se conoce de antemano el número de iteraciones. Se ejecuta mientras la condición sea verdadera. Su sintaxis básica es:

while (condición) {
    // Código a ejecutar
}

Ejemplo: Imprimir números del 1 al 5

int i = 1;
while (i <= 5) {
    std::cout << i << std::endl;
    i++;
}
